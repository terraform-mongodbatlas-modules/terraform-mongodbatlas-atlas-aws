```release-note:breaking-change
variable/privatelink_byo_endpoint: Changes type from `map(string)` to `map(object({ region, supported_remote_regions }))` to support cross-region BYOE PrivateLink
```

```release-note:enhancement
variable/privatelink_byo_service: Adds optional `region` and `service_region_key` fields for cross-region BYOE. A VPC endpoint in one AWS region can reference an Atlas endpoint service in another via `service_region_key`
```
