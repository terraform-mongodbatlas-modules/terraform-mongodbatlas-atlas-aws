```release-note:breaking-change
variable/backup_export: Adds `expiration_days` to `create_s3_bucket` (default 365 days). Module-managed backup S3 buckets now create an `aws_s3_bucket_lifecycle_configuration` resource. Existing users see a non-empty plan adding the lifecycle rule. Set `expiration_days = 0` to opt out and preserve previous behavior
```
```release-note:enhancement
output/backup_export: Adds `expiration_days` to `backup_export` output (365 default, 0 = disabled, null = BYO bucket)
```
